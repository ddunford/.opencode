# Migrate Jest tests to Vitest

## When to use
Use when modernising a TS frontend test stack or standardising tooling.

## Checklist
- Map Jest config to Vitest equivalents
- Replace globals and mocking APIs carefully
- Fix environment and DOM setup
- Update snapshots and timers behaviour
- Run suite in CI and track flaky tests
